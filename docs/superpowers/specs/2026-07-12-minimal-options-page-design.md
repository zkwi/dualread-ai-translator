# 精简设置页设计

## 目标

让普通用户只面对完成连接、选择翻译方向和控制阅读体验所需的设置。低频技术参数继续使用安全默认值，并兼容已有本地存储，但不再出现在界面中。

## 信息架构

### 常用设置

1. API 连接：服务商、模型、API Key、测试 API。
2. 翻译语言：源语言、目标语言。
3. 阅读体验：自动翻译、展示方式。

移除重复的快速开始步骤、手动保存按钮和语言预设按钮。设置仍使用现有防抖自动保存。

### 高级设置

高级区域默认折叠，只保留低频但需要可逆入口的能力：

- 界面语言（默认跟随浏览器）。
- 自定义 API 地址。
- Thinking 自动控制与手动策略覆盖。
- 自定义翻译提示词及恢复默认提示词。
- 同时翻译段落数。
- 清空缓存、恢复全部默认设置。

选择“自定义”服务商时自动展开高级区域，避免隐藏必填的 API 地址。

## 移出界面的参数

- API 超时。
- 成本模式、视口开关。
- 每轮段落数、单段长度、每页请求数和字符数。
- 缓存保留天数和最大条目数。

这些键保留在 `shared.js` 默认值和 Chrome 本地存储中。设置页保存时不写入它们，因此升级不会覆盖已有用户的值。

## 测试

- 常用区域只包含批准的必要控件。
- 高级区域默认折叠，且只包含低频但需要可逆入口的高级能力。
- 自定义服务商会展开高级区域。
- 自动保存、API 测试、Thinking 探测、提示词恢复和维护操作继续工作。
- options 页面在中英文文案下不横向溢出。
